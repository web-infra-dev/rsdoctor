# 介绍

## 什么是 Rsdoctor？

**Rsoctor 是一个面向构建过程与构建产物的诊断与分析的一站式工具**，同时支持多种前端类库在项目中使用时的诊断与分析，来帮助开发者快速发现并解决问题。

你通过使用 Rsdoctor 可以看到更加直观的项目问题诊断信息，以及深入分析更具体的项目问题，比如：

- 更加直观的看到 **重复包**被解析的链路与打包的原因
- 深入分析 Webpack Loader 的**调用过程与优化构建耗时**
- 更加清楚的知道这行代码没有被 Tree Shaking 掉的链路与原因
- ......

而目前我们支持的前端类库主要为以下两类，你可以在下面的库中接入并使用：

- 构建相关：
  - [Rspack](https://www.rspack.dev)
  - [Webpack](https://webpack.js.org)
  - [Modern.js](https://modernjs.dev/builder)
