## enableNativePlugin

- **类型：** `boolean`
- **可选：** `true`
- **默认值：** `false`

通过 enableNativePlugin 选项开启 Rspack 原生插件，将 Rsdoctor 中耗时较大的数据处理逻辑前置到 Rspack 构建阶段，从而显著提升构建分析效率并大幅降低 Rsdoctor 自身的分析耗时。

- **示例：**

```js
new RsdoctorRspackPlugin({
  experiments: { enableNativePlugin: true }
}),
```
